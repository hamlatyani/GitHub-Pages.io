Politique de confidentialité

Dernière mise à jour : 14 avril 2026

1. Introduction

Bib&Butt est une application de suivi quotidien pour bébé, conçue pour les parents. Nous prenons la protection de vos données très au sérieux.

2. Données collectées

L'application collecte uniquement les données que vous saisissez volontairement :

  • Votre prénom (Papa, Maman, etc.)
  • Les événements de suivi : biberons, couches, soins, médicaments, poids, taille
  • Les dates et heures associées
  • Les notes libres que vous ajoutez

Nous ne collectons aucune donnée personnelle d'identification (email, téléphone, adresse), ni donnée de localisation, ni donnée de santé au sens réglementaire.

3. Stockage des données

Les données sont stockées sur Google Firebase (Firestore), hébergé dans l'Union européenne. Elles sont associées à un code famille anonyme (8 caractères aléatoires) et ne sont liées à aucun compte utilisateur identifiable.

4. Partage des données

Les données sont partagées uniquement entre les personnes possédant le même code famille. Aucune donnée n'est vendue, transmise ou partagée avec des tiers, annonceurs ou partenaires commerciaux.

5. Pas de publicité ni de tracking

L'application ne contient aucune publicité, aucun SDK de tracking, aucun analytics tiers. Nous n'utilisons pas de cookies.

6. Suppression des données

Vous pouvez supprimer vos événements directement dans l'application. Pour supprimer l'ensemble de vos données, quittez la famille depuis l'onglet Profil. Si vous souhaitez une suppression complète de vos données côté serveur, contactez-nous.

7. Sécurité

Les communications entre l'application et nos serveurs sont chiffrées via HTTPS/TLS. L'accès aux données est limité aux détenteurs du code famille.

8. Données des enfants

L'application enregistre des données de suivi (repas, couches, poids) mais ne collecte aucune donnée personnelle identifiable de l'enfant (nom complet, photo, date de naissance exacte).

9. Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous à : hamlat.yani@icloud.com

10. Modifications
