Questions frequentes

Comment partager le suivi avec l'autre parent ?

Le premier parent cree un suivi et obtient un code a 8 caracteres. Le second parent ouvre l'app, choisit "Rejoindre un suivi existant" et entre ce code. Les deux parents partagent ensuite les memes donnees en temps reel.

Mes donnees sont-elles securisees ?

Oui. Les donnees sont stockees sur Firebase (Google Cloud) avec chiffrement HTTPS. Seules les personnes possedant votre code famille peuvent acceder a vos donnees. Aucune donnee n'est partagee avec des tiers.

Comment supprimer un evenement ?

Appuyez sur le bouton X rouge sur l'evenement dans la timeline, ou faites un appui long pour acceder au menu avec l'option "Supprimer".

Comment ajouter une action rapide personnalisee ?

Dans la barre d'actions rapides en bas, appuyez sur le bouton "+" a droite du separateur. Choisissez un nom et une icone, puis validez. Votre action apparaitra dans la barre.

Comment voir les evenements d'un jour passe ?

Utilisez les fleches gauche/droite autour de la date en haut de l'ecran, ou appuyez sur la date pour ouvrir le calendrier et choisir n'importe quel jour.

Comment supprimer toutes mes donnees ?

Allez dans l'onglet Profil et appuyez sur "Quitter la famille". Pour une suppression complete cote serveur, contactez-nous par email.
